# 🚀 PayTrack CM — Système de Recouvrement Intelligent

![PayTrack CM Banner](https://github.com/user-attachments/assets/0aa67016-6eaf-458a-adb2-6e31a0763ed6)

PayTrack CM est une plateforme B2B conçue pour automatiser le suivi des créances et accélérer les encaissements via Mobile Money (MTN & Orange) au Cameroun.

## 🌟 Fonctionnalités principales

- **Tableau de Bord Premium** : Visualisation en temps réel du taux de recouvrement et des créances critiques.
- **Paiements MoMo Intégrés** : Génération de liens de paiement Campay (MTN/Orange) pour chaque facture.
- **Relances Automatisées** : Moteur de relance WhatsApp (360dialog) avec escalade intelligente du ton.
- **Gestion Clients** : Base de données centralisée avec calcul automatique du score de crédit.

## 🚀 Démarrage Rapide

### Configuration
1. Clonez le dépôt.
2. Installez les dépendances :
   ```bash
   npm install
   ```
3. Configurez votre fichier `.env` avec vos accès Supabase, Campay et 360dialog.

### Lancement Local
Pour lancer le Frontend (React) et le Backend (Node.js) simultanément :
```bash
npm run dev:all
```
